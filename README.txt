Запуск программы через ui.py

Для добавления пунктов назначения (аэропортов) нужно нажать кнопку Add waypoint и прописать полный путь до json файла с данными о пункте назначения.
В json файле name - название пункта назначения, ways - пути до связанных пунктов в формате 
[["Пункт2", [<минимальное время пути до пункта>, <максимальное время пути до пункта>]], ["Пункт3", [.., ..]] ... ]

Чтобы получилось добавить новый пункт, в ways должны быть связи только с уже существующими в программе пунктами.
(примеры пунктов в Moscow.json, London.json, Berlin.json и New-York.json)
